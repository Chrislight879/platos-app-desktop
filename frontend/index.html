Acerca de la Aplicación

Esta aplicación genera platos de comida balanceados basados en grupos alimenticios.

Características:

  • Generación automática de platos
  • Base de datos con +25 alimentos
  • Sistema de porciones inteligente
  • Información nutricional estimada
  • Funciona completamente offline
  • No requiere instalación de servidores

Versión: 1.0.0

Tecnología: Node.js + SQLite

Base de datos: SQLite (archivo local)
